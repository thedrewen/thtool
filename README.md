## Get Started

**Install :**
```bash
npm i thtool
```

**Setup :**
```bash
npx thtool setup
```

**Server :**
```bash
npx thtool server
```

**Chat :**
```bash
npx thtool chat
```

* Use -p or --port to set the port. (The default value is 8080.)
